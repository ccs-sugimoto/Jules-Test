# テトリスゲーム

シンプルなテトリスゲームです。ウェブブラウザ上で動作します。

## 利用方法

1.  このリポジトリのファイルをダウンロードします。
2.  お使いのウェブブラウザで `index.html` ファイルを開いてください。

## 操作説明

*   **← (左矢印キー)**: ブロックを左に移動
*   **→ (右矢印キー)**: ブロックを右に移動
*   **↓ (下矢印キー)**: ブロックを通常より速く下に移動 (ソフトドロップ)
*   **↑ (上矢印キー)**: ブロックを回転
*   **スペースキー**: ブロックを一番下まで一気に落下 (ハードドロップ)
*   **Enterキー**: ゲームオーバー後にゲームをリスタート
